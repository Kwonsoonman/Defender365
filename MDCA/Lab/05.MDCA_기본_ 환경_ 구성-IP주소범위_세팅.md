# MDCA 기본 환경 구성 – IP 주소 범위 세팅

1.	Microsoft Defender 포탈의 [설정] – [클라우드 앱]에서 [시스템]-[IP 주소 범위] 메뉴를 클릭합니다.
 ![](05.MDCA_기본_%20환경_%20구성-IP주소범위_세팅.assets/2026-06-14-13-58-26.png)

2.	새 IP 주소 범위에서 [이름], [IP주소 범위],[범주],[태그]등의 정보를 입력 후 [만들기]를 클릭합니다.
![](05.MDCA_기본_%20환경_%20구성-IP주소범위_세팅.assets/2026-06-14-13-58-37.png)
