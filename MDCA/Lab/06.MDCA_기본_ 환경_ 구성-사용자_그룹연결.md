# MDCA 기본 환경 구성 – 사용자 그룹 연결

1.	Microsoft Defender 포탈의 [설정] – [클라우드 앱]에서 [시스템]-[사용자 그룹] 메뉴에서 [+사용자 그룹 가져오기]를 클릭합니다.<br>
 ![](06.MDCA_기본_%20환경_%20구성-사용자_그룹연결.assets/2026-06-14-14-00-23.png)

2.	사용자 그룹 가져오기에서 그룹을 가져올 앱에서 [Office 365]를 선택합니다.<br>
 ![](06.MDCA_기본_%20환경_%20구성-사용자_그룹연결.assets/2026-06-14-14-00-34.png)

3.	가져올 사용자 그룹 선택에서 추가할 그룹을 검색하여 추가하고, [가져오기가 완료되면 전자 메일을 알림]을 클릭합니다.<br>
![](06.MDCA_기본_%20환경_%20구성-사용자_그룹연결.assets/2026-06-14-14-00-44.png)
 

4.	사용자 그룹 목록에 추가한 그룹이 추가됩니다.<br>
 ![](06.MDCA_기본_%20환경_%20구성-사용자_그룹연결.assets/2026-06-14-14-00-52.png)
 
5.	되고, 그룹의 구성원을 가져오게 되면 멤버의 숫자가 업데이트 됩니다. <br>
 ![](06.MDCA_기본_%20환경_%20구성-사용자_그룹연결.assets/2026-06-14-14-01-00.png)
