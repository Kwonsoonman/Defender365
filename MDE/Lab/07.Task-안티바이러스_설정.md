# 작업 7. 안티 바이러스 설정

1.	Microsoft Intune 포탈에서 [엔드포인트 보안] – [바이러스 백신]을 클릭합니다. <br>
![](07.Task-안티바이러스_설정.assets/2026-06-14-18-25-12.png) 

2.	프로필 만들기에서 [플랫폼] / [템플릿 – Microsoft Defender 바이러스 백신]을 선택하고 [만들기]를 클릭합니다. <br>
 ![](07.Task-안티바이러스_설정.assets/2026-06-14-18-25-20.png)


3.	정책 만들기 단계에서 [이름], [설명]을 입력합니다 <br>
 ![](07.Task-안티바이러스_설정.assets/2026-06-14-18-25-27.png)


4.	구성 설정 단계에서 안티 바이러스를 위한 설정등을 선택 설정합니다. <br>
 ![](07.Task-안티바이러스_설정.assets/2026-06-14-18-25-35.png)

 
5.	범위 태그 단계에서 관리하려는 태그를 지정합니다.<br>
 ![](07.Task-안티바이러스_설정.assets/2026-06-14-18-25-43.png)

6.	할당 단계에서 안티바이러스 정책을 적용할 대상자를 추가합니다. <br>
 ![](07.Task-안티바이러스_설정.assets/2026-06-14-18-25-50.png)

 
8.	검토+만들기 단계에서 설정된 안티바이러스 설정 내용을 확인 후 저장 합니다.<br>
 ![](07.Task-안티바이러스_설정.assets/2026-06-14-18-25-58.png)

9.	안티 바이러스 정책이 목록에 추가됩니다.<br>
 ![](07.Task-안티바이러스_설정.assets/2026-06-14-18-26-05.png)
